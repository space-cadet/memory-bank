Getting Started with Memory Bank

Get up and running with Memory Bank in under 5 minutes. No complex setup required.

Prerequisites

Quick Setup

1

Install Memory Bank

Run this command in your terminal:

npm install -g memory-bank-cli
2

Create Your First Project

Navigate to your project directory and initialize:

cd your-project mb init
3

Basic Configuration

Set up your basic configuration:

mb config --init

Verify Installation

Run a Quick Test

Create your first task to verify everything is working:

mb task create "My first task"

Expected Output:

✓ Task created successfully
Task ID: T1
Status: Active
Location: memory-bank/tasks/T1.md
